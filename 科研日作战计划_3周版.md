## 科研日常见问题 Q&A 准备

| 问题 | 建议回答方向 |
|------|-------------|
| 为什么选 FinBERT？ | 专门为金融文本训练的BERT变体，比通用情感工具更懂金融术语；实验也证明了这一点（AAPL专项对比） |
| 为什么选 LSTM？ | 经典时序模型，适合捕捉股价的时间依赖关系 |
| 为什么选这三只股票？ | 代表不同行业（科技/电动车/电商），波动特征各异，有代表性 |
| AAPL时间范围为何更长？ | AAPL新闻数据最丰富（91万条），完整覆盖2020-2025；TSLA/AMZN受数据可获取性限制 |
| 融合模型提升了多少？ | 看实验结果，通常 RMSE 降低 5-15%，方向准确率提升 3-8% |
| 能用来实际炒股吗？ | 这是学术研究，实际交易还需考虑交易成本、滑点、市场冲击等因素 |
| 局限性是什么？ | TSLA/AMZN数据范围较短、新闻覆盖有噪声、未考虑宏观经济因素 |
