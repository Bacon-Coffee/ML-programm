## 关键超参数

| 参数 | 建议值 | 说明 |
|------|--------|------|
| 滑动窗口 | 30天 | 用过去30天预测下一天 |
| LSTM hidden_size | 64 或 128 | 4060 完全够用 |
| LSTM num_layers | 2 | 多了容易过拟合 |
| batch_size | 32 | |
| learning_rate | 0.001 | Adam 优化器 |
| epochs | 100 | 配合早停法(patience=10) |
| dropout | 0.2 | 防止过拟合 |
| 训练/验证/测试 | 70/15/15 | 按时间顺序切分 |

## 科研日常见问题 Q&A 准备

| 问题 | 建议回答方向 |
|------|-------------|
| 为什么选 FinBERT？ | 专门为金融文本训练的BERT变体，比通用BERT更懂金融术语 |
| 为什么选 LSTM？ | 经典时序模型，适合捕捉股价的时间依赖关系 |
| 为什么选这三只股票？ | 代表不同行业（科技/电动车/电商），新闻覆盖量大，波动特征各异 |
| 融合模型提升了多少？ | 看实验结果，通常 RMSE 降低 5-15%，方向准确率提升 3-8% |
| 能用来实际炒股吗？ | 这是学术研究，实际交易还需考虑交易成本、滑点、市场冲击等因素 |
| 局限性是什么？ | 新闻覆盖不完整、情感分析有噪声、未考虑宏观经济因素 |
